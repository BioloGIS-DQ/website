Acerca de BioloGIS

¿De qué trata este blog?

El blog de BioloGIS esta pensado como un espacio de aprendizaje y descubrimiento para todos. Este es un sitio que me permite compartir, enseñar y dialogar con personas interesadas como tú sobre el mundo de los Sistemas de Información Geográfica (SIG). Este será un espacio de aprendizaje que permita aterrizar con ejemplo concretos, prácticos, pero sobre todo, sencillos para que la gente pueda aprender. Todo lo anterior, con el ánimo de compartir conocimiento, experiencias y saberes que considero son esenciales para los biólogos y profesionales de hoy en día. ¡Bienvenido y Bienvenida 😁!

¿Para quién es este blog?

Este blog está dirigido a interesados en el mundo SIG, desde principiantes hasta veteranos. El mundo GIS es tan grande que permite el encuentro e interacción de múltiples disciplinas.

¿Dónde puedo trabajar con GIS?

Hoy día hay varias soluciones de software libre y abierto (Open source) que permiten explorar los GIS de manera asequible. Precisarás de una computadora y buena voluntad. Las especificaciones técnicas de cada equipo son muy variadas, más en mi experiencia, tareas elementales GIS usando sets de datos no tan grandes, pueden ejecutarse bien incluso en equipos muy modestos y algo antiguos.

¿Qué opciones de software GIS existen?

Existen múltiples aplicaciones, librerías y soluciones de software para manejar y procesar datos GIS. Algunas creadas para tareas multipropositos generales, y otras para tareas y aplicaciones específicas. Algunas de las principales son:

Software Libre

  • QGIS
  • GRASS
  • SAGA GIS
  • GDAL*
  • Python
    • geotools
  • R
    • geotools
  • Google Earth Engine^ - GEE

Software Propietario

  • ArcGIS Pro
  • ArcMap
  • CartoDB

*Paquetes y librearías embebidos dentro de otras aplicaciones

^Requiere registro y creación de una cuenta

¿Dónde puedo conseguir Qgis?

El proyecto Qgis esta disponible en línea listo para descargar a distintas plataformas y Sistemas operativos al visitar el sitio web oficial. Qgis es un proyecto que se actualiza constantemente gracias al los múltiples esfuerzos de muchos desarrolladores, usuarios y contribuidores alrededor del mundo. Hay muchas maneras de instalar Qgis en un equipo, más podrás encontrar dos distribuciones típicas.

Instaladores dedicados de Qgis:

Puedes encontrar los instaladores en la página oficial del proyecto. Recuerda que existe versiones:

  • Última versión (Latest version) estable publicada
  • Versión estable de largo plazo - Long Term Version - LTR

Dependiendo de tus necesidades y familiaridad con el software puedes escoger la que más te convenga.

Instaladores Web (OSGeo4W)

Si necesitas usar Qgis y sus dependencias actualizadas, una buena opción es utilizar el Instalador web de OSGeo4W, de esta manera podrás instalar y actualizar Qgis y sus librerías de manera sencilla y eficaz.

Nota: Esta opción no es recomendada para usuarios muy nuevos.

Si quieres saber aprender a usar este instalador mira mí guía al respecto: Instalando Qgis con OSGeo4W
