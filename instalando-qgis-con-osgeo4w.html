Instalando Qgis con OSGeo4W

¿Cómo instalar Qgis usando OSGeo4W?

Instalar Qgis es realmente sencillo, existen múltiples maneras de realizarlo. Sin embargo, cada una presenta sus ventajas y desventajas.

Instalando Qgis como un binario (Stand-alone)

Desde la página de descargas oficial de Qgis puedes obtener un instalador (PC/MAC) que tiene todo lo necesario para correr Qgis en tu maquina. Basta con solo correrlo, especificar donde instalar y listo! Tienes Qgis en tu equipo!

Sin embargo, esta forma de instalación te instala una versión específica de Qgis, la que esté en el momento.

Las versiones de Qgis

Si no conocías, Qgis es un software que se actualiza constantemente, por lo menos una vez al mes, gracias a los esfuerzos de múltiples desarrolladores y usuarios que constantemente buscan afinar y mejorar esta gran proyecto.

Cada cierto tiempo una nueva versión de Qgis, con nuevos ajustes, herramientas, o parches para corregir errores, es publicada. Cada versión busca mejorar desempeño y calidad de la experiencia con los usuarios en pequeños y grande detalles. Sin embargo, si en tu flujo de trabajo no has requerido de las últimas versiones, y trabajas con una versión que te satisface, con el tiempo puede ocurrir que termines trabajando en una versión que ha quedado sin soporte. Si en algún momento quieres utilizar algún plugins reciente o funcionalidad novedosa, será necesario que actualices tu versión de Qgis. De lo contrario, si fuerzas usar plugins en versiones no soportadas, no hay garantía de que funciones dado que la versión de tu maquina puede no disponer las dependencias requeridas. 👀 No es necesario tener la última versión de Qgis para poder trabajar muy bien, pero si buscas trabajar con las ultimas funcionalidades y estabilidad es esencial mantenerte al día.

Una manera de ir 'seguro' es trabajar con una versión particular: la versión de soporte extendido (Long Term Resolution - LTR), que está optimizada para estabilidad, evitar errores y estará constantemente soportada y mantenida durante 1 año.

EL Problema de mantener todo actualizado

Si instalaste alguna vez Qgis con los instaladores binarios (Standalones), no tienes una manera directa para actualizar a una nueva versión de Qgis. Qgis te advertirá que hay una nueva versión disponible, pero si intentas usar el mismo procedimiento con los instaladores dedicados, terminaras por tener dos copias de Qgis distintas en tu maquina. Para muchos usuarios, este no es el escenario deseado.

Aparte de todo, si tu flujo de trabajo involucra múltiples complementos y plugins, en ocasiones las librerías y dependencias con los que trabajan también pueden cambiar de versión y actualizarse, lo que provoca que termines con mas de una versión del mismo software, o potencialmente generar combinaciones incompatibles entre sí.

Para los que desean siempre trabajar con la última versión puede sonar confuso y terrible, pero, realmente es un problema que tiene fácil solución.

La solución: Instalador OSGeo4W

Previniendo las situaciones anteriores, existe una manera dinámica que permite instalar Qgis y software complementario (GRASS, SAGA, algunas dependencias) de manera dinámica, sencilla y de fácil administración para actualizaciones. Incluso permitiendo eliminar o retroceder a versiones antiguas si se requiere.

El instalador OSGeo4W es un instalador web desarrollado por la [Open Source Geospatial Foundation (OSgeo)](https://www.osgeo.org/ la ONG que creó, patrocina y mantiene un conjunto de software geoespacial libre (como Qgis, GRASS GIS, GDAL, PostGIS, GeoServer, OpenLayers, entre muchos otros).

Utilizar este instalador permite maneter atucxliazado y organizado el fopwatrew geoepacil en tu maquina de una manera seniclla, direacta y sistemátcia. Recomednado para usaurios SIG constnaes. Si bien puede parece complejo su ocnfiguración inicial, en esta guía te mostraré como manejar tus inatakicoens de Qgis a travez de este aplciativo.
